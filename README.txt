A simple expert system which automatically detects what scale to improvise over as well as the arpeggios played over chords that are not in the scale. Run in a LISP environment.

Batch JGI.clp to run the Jazz Guitar Improvisation Expert System.
The other files are batched from JGI.clp. The files included in this expert
system are

JGI.clp
JGIChordBuilds.clp
JGIFunctions.clp
JGIGlobals.clp
JGIRules.clp
